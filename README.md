##Requires

- node7.0.0
- php >=7.1

To install php7.2:
```sh
$ sudo add-apt-repository ppa:ondrej/php
$ sudo apt-get update
(optional) $ sudo apt-get remove php7.0
$ sudo apt-get install php7.2
```

PPA [link](https://launchpad.net/~ondrej/+archive/ubuntu/php/).
